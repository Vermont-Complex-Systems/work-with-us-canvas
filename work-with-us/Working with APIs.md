Application Programming Interfaces (APIs) facilitate exchange information between teams, both within and across organizations. [[Open standards]] such as [[openAPI]] have emerged as solution to standardize this exchange of information across various stakeholders, describing inputs and outputs of API ([[ponelat_designing_2022]]).
